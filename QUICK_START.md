# ⚡ QUICK START - 10 MINUTOS PARA O AR!

## 🎯 Objetivo
Colocar seu app Streamlit online em 10 minutos.

---

## ✅ MÉTODO 1: Deploy Direto (SEM INSTALAR NADA!)

### 1. GitHub (3 min)
1. https://github.com → Sign up
2. New repository → `analise-deslizamentos`
3. Upload: `streamlit_app.py`, `requirements.txt`, `README.md`

### 2. Streamlit Cloud (2 min)
1. https://share.streamlit.io → Sign in with GitHub
2. New app → Selecionar seu repositório
3. Deploy!

### 3. Pronto! (5 min)
Sua URL: `https://xxx.streamlit.app`

---

## ⚙️ MÉTODO 2: Testar Localmente Primeiro

### Windows:
```bash
pip install streamlit pandas plotly numpy
streamlit run streamlit_app.py
```

### Mac/Linux:
```bash
pip3 install streamlit pandas plotly numpy
streamlit run streamlit_app.py
```

Abre em: http://localhost:8501

## 🆘 Problemas? Soluções Rápidas

**App não inicia:**
→ Reboot app no Streamlit Cloud

**Erro de módulo:**
→ Verifique requirements.txt no repositório

**Gráficos não aparecem:**
→ Faça upload dos CSVs novamente

**Deploy travou:**
→ Aguarde 5min ou delete e recrie
